**Models**
==========

:math:`\text{Colibri}` is a tool which allows to define and use any model providing theory predictions comparable to measurements. It has two models implemented at the moment:

Model list:
===========

- ``wmin``: the weight minimisation model
- ``grid_pdf``: the grid PDF model
